Аудиозаписи - простой аудиоплеер, работающий на технологии flash. Проигрывает mp3 файлы, которые были загружены в топик с помощью плагина Attachments (http://livestreetcms.com/addons/view/79/) или если в топике есть ссылки (!) на аудиофайлы.

Также позволяет загружать в топики mp3 файлы, для которых будут показаны плееры.

Плагин использует сервис playmp3.org.ua.

Кнопка загрузки mp3 файлов на панели доступна только для редакторая MarkItUp, который установлен по-умолчанию!
